## 🐛 解决的问题

### 问题1：环境变量未加载
**原因：** `.env` 文件在 `backend/` 目录，但Python从项目根目录运行

**解决方案：** 在项目根目录创建 `.env` 文件，包含：
```bash
DB_NAME_TO_USE=aliyun
ZHIPU_API_KEY=0952801ba29f4365b9298fc0fc3b9e80.OlryWE4QUP8Surq4
ZHIPU_MODEL=glm-4-plus
API_HOST=0.0.0.0
API_PORT=8000
```

### 问题2：默认数据库配置
**原因：** 环境变量为空时，代码默认使用 `'aliyunDB'`，但配置文件中是 `'Aliyun DB'`

**解决方案：** 确保 `.env` 文件中的 `DB_NAME_TO_USE=aliyun` 正确配置

### 问题3：前端依赖缺失
**原因：** `node_modules` 不存在

**解决方案：** 运行 `npm install` 安装依赖（耗时约1分钟）

## 🐛 常见问题排查

### 问题1：前端显示"无法加载Dashboard指标"
**可能原因：**
- 后端服务未启动
- 数据库连接失败
- 预计算表不存在

**解决方法：**
```bash
# 检查后端服务
curl http://localhost:8000/api/v2/

# 检查数据库连接
python -c "from backend.database import Database; db = Database(); print(db.execute_query('SELECT 1'))"
```

### 问题2：买家列表为空
**可能原因：**
- 数据库中没有数据
- 预计算表未生成

**解决方法：**
```bash
# 运行预计算刷新
mysql -u username -p database_name -e "CALL refresh_target_buyers_precomputed();"
```

### 问题3：CORS错误
**可能原因：**
- 前端和后端端口不同
- 后端未配置CORS

**解决方法：**
检查 `backend/main.py` 是否包含CORS中间件：
```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

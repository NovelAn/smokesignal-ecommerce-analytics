## 📈 版本对比

### v1.0 vs v2.0

| 特性 | v1.0 (已废弃) | v2.0 (当前) |
|------|--------------|-------------|
| 数据源 | 直接查询视图 | 预计算表 |
| 客户范围 | 全部买家 | 目标买家(391人) |
| 响应时间 | 10-30秒 | 0.1-0.3秒 |
| API版本 | /api/v1 | /api/v2 |
| 性能提升 | 基准 | **40-120x** |
| 自动更新 | ❌ | ✅ 每天11:00 |

**废弃原因：**
- v1.0使用buyer_summary表方案已过时
- 查询性能无法满足需求
- 目标客户范围不够明确
